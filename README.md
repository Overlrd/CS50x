### Plurality 
A program that runs a plurality election, per the below.
```
$ ./plurality Alice Bob Charlie
Number of voters: 4
Vote: Alice
Vote: Bob
Vote: Charlie
Vote: Alice
Alice
```
 - link: [here](https://cs50.harvard.edu/x/2023/psets/3/plurality/)

### readability

